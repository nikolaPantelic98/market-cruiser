Welcome to MarketCruiser

Since this is my personal project and not an actual online store (yet), I would like to share with you the details of this project.

MarketCruiser is an eCommerce project developed using Java, Spring, Spring Boot, Spring Security, Thymeleaf, jQuery, AJAX, and MySQL. It aimed to provide a comprehensive online shopping platform with user management, category and brand management, product management, customer management, order management, and more.

Highlights:

•  Developed a multi-module Maven project, comprising server-side, client-side, and common code modules, ensuring modularity and scalability of the application.

•  Implemented robust user authentication and authorization using Spring Security, ensuring secure access to the system and role-based permissions for different user roles.

•  Designed and developed efficient category and brand management features, enabling administrators to perform CRUD operations, manage subcategories, and export category data to CSV format, facilitating seamless organization and navigation of products.

•  Developed the product management module, enabling administrators to efficiently manage product information, including CRUD operations, pagination, sorting, and filtering, resulting in improved product catalog management.

•  Implemented customer registration and authentication functionalities, incorporating email verification for secure account creation, password reset via email, and support for login via Gmail, streamlining the customer onboarding process.

•  Developed a dynamic shopping cart system, allowing customers to add products, adjust quantities, and calculate total prices accurately, enhancing the overall shopping experience and facilitating smooth checkout processes.

•  Implemented comprehensive order management features for both administrators and customers, including order tracking, order status management, returns, and order listing, ensuring efficient handling of orders.

•  Integrated multiple payment methods, such as Cash on Delivery (COD) and PayPal, providing customers with flexible and secure payment options.

•  Successfully deployed the server-side and client-side applications on Heroku, ensuring high availability and scalability of the platform, while effectively managing the migration of images and files to AWS Amazon S3 for optimized performance and storage.
